Welcome to the CowoBox website

The CowoBox is a very easy to use device. Like the PirateBox but a with more powerjuice. It hosts coworking applications (etherpad, ...) to speed up working group in remote no-internet place.

3 easy steps to start with the CowoBox:

  1. Connect to the "COOWOBOX" wifi network
  2. Open your web browser
  3. Enter any URL and you will be redirected to the CowoBox welcome page (etherpad for now).

How-to build your own CowoBox

Let's grab a nifty shiny single board computer, they have been very popular since the RaspberryPi marketing, but any board will do (Olinuxino, GooseBerry, BeagleBoard, etc). Olimex (not affiliated) do some nice board with LiPo battery charger included (that's a plus):

Then find a cheap usb wifi dongle compatible with hostapd: hostapd home page Realtek 8192cu are cheap on AliExpress or DX

Install your favorite Unix derivated OS (Debian, Archlinux, FreeBSD, etc ) and add etherpad, redis, dnsmasq, hostpad in the mix. The key component is dnsmasq which reply the wifi ip address to any dns resquest. Example configuration files are on the github for your convinience.
