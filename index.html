Welcome to the CowoBox website

The CowoBox is a very easy to use device. Like the PirateBox but a with more powerjuice. It hosts coworking applications (etherpad, ...) to speed up working group in remote no-internet place.

3 easy steps to start with the CowoBox:

  1. Connect to the "cowobox" wifi network
  2. Open your web browser
  3. Enter any URL and you will be redirected to the CowoBox welcome page (etherpad for now).
